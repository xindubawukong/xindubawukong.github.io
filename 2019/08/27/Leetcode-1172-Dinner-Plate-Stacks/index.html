Leetcode 1172. Dinner Plate Stacks

2019/08/27 Algorithm
  • leetcode

题目

https://leetcode.com/problems/dinner-plate-stacks/

做法

用一个stack的数组维护所有栈，一个set维护所有非空的栈，一个set维护所有没有满的栈即可。每次操作时间复杂度O(logn)。
